# Quantum Machine Learning

A Python repository containing simple simulators and implementations of well-known quantum algorithms.

## Current contents

- `Deutsch_Jozsa.py` — implementation of the Deutsch–Jozsa algorithm
- `Grover.py` — implementation of Grover’s search algorithm
- `Shor.py` — implementation/simulation of Shor’s algorithm
- `README.md` — project documentation
- `LICENSE` — repository license

## About this repository

This project is a collection of Python programs for exploring foundational quantum algorithms in a simple and accessible way. It is useful for:

- learning the basic logic behind quantum computation
- understanding how major quantum algorithms work step by step
- experimenting with algorithmic ideas in Python
- building a base for more advanced quantum computing or quantum machine learning projects

## Algorithms included

### Deutsch–Jozsa Algorithm
The Deutsch–Jozsa algorithm determines whether a Boolean function is constant or balanced with fewer evaluations than a classical method would typically require.

### Grover’s Algorithm
Grover’s algorithm provides a quadratic speedup for unstructured search problems. It is one of the most famous examples of a quantum advantage in search.

### Shor’s Algorithm
Shor’s algorithm is a landmark quantum algorithm for integer factorisation. It is important because it shows that a quantum computer can solve certain number-theoretic problems much faster than known classical methods.
